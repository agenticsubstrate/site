# agenticsubstrate.org site

Static machine-first surfaces for the agentic reputation substrate, a research
instrument operated by Agentic Substrate Research. Deployed via Cloudflare Pages,
output directory: public/. Pre-launch mode per the discoverability spec: all
surfaces truthful, registration not yet open, status at /status.json.

This repository receives generated artifacts from the project's surface
generator; edit upstream, deploy here. Bootstrap version 0.1, 2026-06-11.
